# Portfolio - Braham Zapata

## Problem Statement
Añadir sección de certificados al portafolio en la página About, dividida en Frontend y Backend, mostrando nombre e imagen de cada certificado.

## Architecture
- **Framework**: React 18 con React Router
- **Styling**: Tailwind CSS + shadcn/ui components
- **Theme**: Dark mode (#0F0F0F, #1A1A1A) con acentos naranja (#F5A623)

## Implemented Features (Jan 31, 2026)
- ✅ Sección "My Certificates" en About page
- ✅ Subsección Frontend Development con 2 certificados
- ✅ Subsección Backend Development con 2 certificados
- ✅ Modal para ver certificados en grande
- ✅ Diseño responsive (desktop, tablet, mobile)
- ✅ Efectos hover y animaciones

## Core Requirements
- Diseño consistente con el resto del portafolio
- Dividido en categorías Frontend/Backend
- Imagen y nombre para cada certificado
